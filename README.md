## Описание проекта

Daily Budget Manager — это веб-приложение для планирования личного бюджета
на заданный период с учётом ежедневных расходов.

Пользователь задаёт стартовый баланс и срок бюджета, после чего приложение
рассчитывает дневной лимит и автоматически перерасчитывает доступную сумму
на каждый день с учётом уже потраченных средств.

Проект демонстрирует работу с датами, локальным хранилищем браузера,
валидацию данных и динамический рендеринг интерфейса без использования
фреймворков.

---

## Используемые технологии

- TypeScript
- Vite
- Tailwind CSS
- IndexedDB (idb)
- date-fns
- Zod
- ESLint
- Prettier

---

## Функциональность

- создание бюджета (баланс и период);
- расчёт дневного бюджета
- добавление ежедневных расходов;
- история расходов;
- удаление расходов;
- редактирование бюджета;
- сохранение данных в IndexedDB;
- адаптивный интерфейс (mobile / tablet /desktop).

## Установка и запуск проекта

Клонировать репозиторий:

```bash
git clone https://github.com/smirnov-dev-frontend/higher-web-practice-calculator/.git
```
## Установка зависимостей
```bash
npm install
```
## Запуск в режиме разработки
```bash
npm run dev
```
## Сборка для продакшена
```bash
npm run build
```
